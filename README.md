# GUAW GUARDIANS

Project Setup for GUAW SaaS Platform.
Includes Landing Page (Marketing) and Console (Dashboard).

## Tech Stack

- React + Vite
- Tailwind CSS
- Framer Motion
- React Router DOM

## Directory Structure

- `src/pages/landing`: Public marketing pages.
- `src/pages/dashboard`: Private dashboard pages.
- `src/components`: UI components and layout.

## Running Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
